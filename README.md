# 유튜브 맛집 댓글 텍스트 분석 프로젝트

유튜브 맛집 추천 채널의 댓글 데이터를 분석하여 어떤 음식 카테고리가 가장 많은 좋아요를 받는지, 댓글에서 자주 등장하는 키워드는 무엇인지 등을 탐색하는 데이터 분석 프로젝트입니다.

댓글의 좋아요 수, 음식 카테고리, 날짜 정보를 활용한 탐색적 데이터 분석(EDA)과 자연어 처리 기반 키워드 분석을 수행합니다.

---

# 프로젝트 목표

다음과 같은 질문에 답하기 위해 데이터를 분석합니다.

- 좋아요(likes)가 100개 이상인 댓글은 어떤 특징이 있을까?
- 어떤 음식 카테고리가 가장 많은 좋아요를 받을까?
- 좋아요가 많이 발생한 날짜는 언제일까?
- 댓글에서 자주 등장하는 단어는 무엇일까?

---

# 데이터셋

`korean_youtube_food_comments.csv`

| 컬럼 | 설명 |
|---|---|
| review | 유튜브 댓글 내용 |
| likes | 댓글 좋아요 수 |
| category | 음식 카테고리 (한식, 양식, 일식 등) |
| start | 댓글 수집 시작 날짜 |
| end | 댓글 수집 종료 날짜 |
| answered | 대댓글 여부 |
